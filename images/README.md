# Folder untuk Foto Produk Kerupuk Atom

Tempat ini adalah folder untuk menyimpan semua foto produk Kerupuk Atom yang akan ditampilkan di website.

## Foto yang Diperlukan

### 1. Logo Website
- **File**: `logo.png`
- **Ukuran**: 40px tinggi (lebar otomatis)
- **Tempat**: Di navbar/header website
- **Format**: PNG atau format lain yang support transparansi

### 2. Foto Produk (untuk Carousel)

Setiap foto akan menampilkan produk dari sudut pandang yang berbeda:

| File | Deskripsi | Ukuran Rekomendasi |
|------|-----------|-------------------|
| `product-1.jpg` | Kemasan Produk | 450x450px atau lebih |
| `product-2.jpg` | Tampilan Depan | 450x450px atau lebih |
| `product-3.jpg` | Sudut Samping | 450x450px atau lebih |
| `product-4.jpg` | Tampilan Atas | 450x450px atau lebih |
| `product-5.jpg` | Dekat / Detail | 450x450px atau lebih |
| `product-6.jpg` | Tekstur Renyah | 450x450px atau lebih |

### 3. Foto Fasilitas Produksi
- **File**: `factory.jpg`
- **Ukuran**: 600x500px atau lebih (landscape)
- **Tempat**: Di bagian "Tentang Kerupuk Atom" (about section)
- **Deskripsi**: Foto fasilitas/pabrik produksi kerupuk atom

## Cara Menambahkan Foto

1. **Siapkan foto Anda** dalam format JPG, PNG, atau WebP
2. **Rename file** sesuai dengan nama di atas (product-1.jpg, product-2.jpg, dll)
3. **Upload file** ke folder `images/` di project ini
4. **Refresh website** untuk melihat hasilnya

## Ukuran File Optimal

- **Format**: JPG untuk foto, PNG untuk logo
- **Ukuran**: Minimal 450x450px (minimal)
- **Maksimal**: 1200x1200px (untuk kualitas lebih baik)
- **File Size**: Usahakan di bawah 500KB per foto untuk loading lebih cepat

## Tips Fotografi Produk

- **Lighting**: Gunakan cahaya yang merata dan terang
- **Background**: Gunakan background solid atau semi-transparent untuk fokus ke produk
- **Angle**: Pastikan setiap sudut menunjukkan detail produk dengan jelas
- **Konsistensi**: Usahakan semua foto memiliki gaya dan lighting yang sama
